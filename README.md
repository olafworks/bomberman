# ボンバーマン

Pygameを使用して作成したボンバーマンゲームです。

<img width="796" alt="スクリーンショット" src="https://github.com/user-attachments/assets/e1854384-118a-4adf-88f7-c9d60749e199" />


## 必要条件

- Python 3.x
- Pygame 2.6.1

## セットアップ方法

1. 必要なパッケージをインストール:
```bash
pip install -r requirements.txt
```

2. ゲームの実行:
```bash
python main.py
```

## 操作方法

- 矢印キー: プレイヤーの移動
  - ↑: 上に移動
  - ↓: 下に移動
  - ←: 左に移動
  - →: 右に移動
- スペースキー: 爆弾を設置
